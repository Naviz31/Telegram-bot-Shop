# Telegram bot shop

`Telegram bot shop` это Telegram-бот-магазин, написанный на Python с использованием `aiogram`.

Проект сделан как портфолио и всё ещё находится в стадии `WIP`, но в нём уже реализован основной сценарий покупки: каталог, корзина, оформление заказа, расчёт доставки, оплата и обработка заказов менеджером.

## Возможности

- Каталог товаров внутри Telegram
- Корзина и быстрая покупка
- Расчёт доставки через Почту России
- Оплата через Telegram
- Создание заказов и обновление их статусов
- Простая панель менеджера для обработки заказов

## Стек

- Python
- Aiogram
- SQLAlchemy
- SQLite

## Запуск

1. Установите зависимости проекта
2. Заполните необходимые значения в `config.py`
3. Запустите бота:

```bash
python run.py
```

## Статус

Этот репозиторий является учебным и портфолио-проектом, а не готовым production-магазином.
